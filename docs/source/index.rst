Tsdat
=====

.. image:: https://zenodo.org/badge/DOI/10.5281/zenodo.5224641.svg
   :target: https://doi.org/10.5281/zenodo.5224641 

**Tsdat** is an open-source Python framework that makes creating pipelines to 
process and standardize time-series data more easy, clear, and quick to stand up
so that you can spend less time data-wrangling and more time on data analysis.
